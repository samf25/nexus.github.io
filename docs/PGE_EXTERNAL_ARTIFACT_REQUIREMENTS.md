# Practical Guide External Artifact Requirements

This file is the source of truth for external artifact prerequisites in `PGE02` to `PGE04`.

## PGE02 - Siege of the Last Gate

Required artifacts in this story:
- `Westwall Ram`
- `Oathbreaker Bell`
- `Sunforge Powder`

Gated options/branches:
- `PGE02-02A / p2-2a-ram` - second-line anchor (`Westwall Ram`)
- `PGE02-02A / p2-2a-bell` - deserter recall and witness pressure (`Oathbreaker Bell`)
- `PGE02-03A / p2-3a-powder` - second-beat reset (`Sunforge Powder`)
- `PGE02-03B / p2-3b-bell` - killbox enforcement (`Oathbreaker Bell`)
- `PGE02-03C / p2-3c-open` - culvert route opening (`Westwall Ram`)
- `PGE02-03C / p2-3c-powder` - extraction decoy (`Sunforge Powder`)
- `PGE02-06 / p2-6-ram` - last-gate brace (`Westwall Ram`)
- `PGE02-06 / p2-6-bell` - oath panic turn (`Oathbreaker Bell`)
- `PGE02-06 / p2-6-powder` - delayed third-turn detonation (`Sunforge Powder`)
- `PGE02-W2` - villain-reversal ending (`Oathbreaker Bell`, `Sunforge Powder`)
- `PGE02-W3` - culvert extraction ending (`Sunforge Powder`, `Westwall Ram`)

## PGE03 - Winter Court Knife-Game

Required artifacts in this story:
- `Mirror of Nine Lies`
- `Green Wax Seal`
- `Veiled Signet`

Gated options/branches:
- `PGE03-01 / p3-1-ledger` - legal entry (`Green Wax Seal`)
- `PGE03-02A / p3-2a-mirror` - decoy exposure (`Mirror of Nine Lies`)
- `PGE03-02B / p3-2b-proof` - witness-anchored petition (`Green Wax Seal`)
- `PGE03-02B / p3-2b-signet` - private testimony access (`Veiled Signet`)
- `PGE03-03 / p3-3-seal` - procedural lock (`Green Wax Seal`)
- `PGE03-03 / p3-3-mirror` - public lie exposure (`Mirror of Nine Lies`)
- `PGE03-04 / p3-4-triad` - triad proof chain (`Mirror of Nine Lies`, `Green Wax Seal`, `Veiled Signet`)
- `PGE03-06 / p3-6-w1` - Archer route finish requires mirror support (`Mirror of Nine Lies`)
- `PGE03-06 / p3-6-w2` - legal route finish requires full triad (`Mirror of Nine Lies`, `Green Wax Seal`, `Veiled Signet`)
- `PGE03-W3` - crowd route finish requires court credentials (`Veiled Signet`, `Green Wax Seal`)
